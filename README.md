# 🎵 AutoComposer

AutoComposerは、Ollama (LLM) による高度な作詞・作曲プロンプト生成と、ACE-Step 1.5 (DiT) による高音質な楽曲生成をシームレスに連携させた、デスクトップ完結型の全自動作曲アプリケーションです。

## 🌟 主な機能

* **完全自動の作詞・作曲フロー**
  * デスクトップの `song_theme.txt` に「夜のドライブに合う曲」など簡単なテーマを書くだけで、Ollamaが「音楽構成プロンプト（英語）」と「歌詞（日本語・ひらがな）」を自動生成し、そのままACE-Stepに渡して曲を生成します。
* **3つの強力な動作モード**
  1. **新曲生成モード**: 全く新しい曲をゼロから作ります。
  2. **再生成モード (アレンジ固定: 歌詞のみ修正)**: 一度作った曲の「メロディーとアレンジ」を完全に維持したまま、歌詞のひらがな（発音）だけを修正して歌い直させることができます。
  3. **再生成モード (アレンジランダム: 歌詞のみ修正)**: 歌詞はそのままに、全く新しいメロディーとアレンジを作り直します。
* **選べるAIモデルと音質**
  * SFTモデル（高音質・緻密なプロ品質） / Turboモデル（高速・軽快なプロトタイプ用）
  * 16bit / 24bit / 32bit Float オーディオ出力対応

## 💻 動作環境

* macOS
* Apple Silicon (M1/M2/M3/M4) 推奨
* 32GB 以上の RAM（LLMとDiTを同時稼働させるため）

## 🚀 クイックスタート

1. デスクトップにある `song_theme.txt` を開き、作りたい曲のテーマや情景（例：「雨の日の午後、カフェで一人で聴くようなジャズ」）を日本語で書き、保存します。
2. デスクトップの `AutoComposer` アプリ（アイコン）をダブルクリックします。
3. 画面の指示に従って、以下の3つを選択します：
   * 動作モード（新曲を作るか、過去の曲を再生成するか）
   * AIモデル（SFT / Turbo）
   * 書き出し音質（16bit / 24bit / 32bit）
4. Terminalが立ち上がり、Ollamaでの作詞・作曲プロンプト生成 → LLMメモリ解放 → ACE-Stepでの楽曲生成 が全自動で進行します。
5. 完成したオーディオファイル、歌詞テキスト、メタデータ(JSON)がデスクトップに出力されます。
