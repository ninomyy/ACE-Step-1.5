# 📘 AutoComposer 取扱説明書 (User Manual)

本アプリは、デスクトップ上の `song_theme.txt` を編集し、`AutoComposer` アイコンをダブルクリックするだけで、Ollama（AI作詞・作曲家）と ACE-Step（AI演奏家）が連携してプロ品質の楽曲を全自動で生成する画期的なツールです。

---

## 🚀 基本的な使い方（新曲生成モード）

1. **テーマを書く**
   デスクトップにある `song_theme.txt` をテキストエディタで開きます。
   作りたい曲のテーマ、情景、ジャンルなどを日本語で自由に書き込みます。
   *(※書き方のコツは後述の「黄金法則」を参照してください)*

2. **アプリを起動**
   デスクトップの `AutoComposer.app` をダブルクリックします。

3. **ダイアログで設定を選ぶ**
   - **動作モード**: `1. 新曲生成モード` を選びます。
   - **モデル**: `SFT`（高音質）または `Turbo`（高速）を選びます。
   - **音質**: `16bit`, `24bit`, `32bit` から選びます。

4. **完成を待つ**
   Terminalが起動し、自動的に作詞と作曲が行われます。
   完了すると、デスクトップに以下の3つのファイルが出力されます。
   - `Song_〇〇_日時_1.flac/wav` (楽曲データ)
   - `Lyrics_〇〇_日時_1.txt` (歌詞とプロンプト)
   - `Song_〇〇_日時_1.json` (メタデータ)

---

## 🔄 再生成モードの使い方

「曲のメロディーやアレンジは最高なのに、歌詞のひらがな（読み方）が一部間違っている！」
そんな時に活躍するのが2つの**「再生成モード」**です。

### 🔧 準備（共通）
1. 直前に出力された `Lyrics_〇〇.txt` を開きます。
2. 「■ 歌詞（ひらがな - ACE-Step 歌唱入力用）」の部分を正しい読みに修正して上書き保存します。
3. デスクトップの `song_theme.txt` を開き、中身をすべて消して **修正した `Lyrics_〇〇.txt` のファイル名（またはパス）** を貼り付けて上書き保存します。

### 🎵 モード2: 再生成モード (アレンジ固定)
歌詞だけを修正し、メロディーや楽器のアレンジは**前回と全く同じものを再現**したい場合はこのモードを選んでください。
アプリが前回のメタデータ（JSON）から「シード値」を読み出し、完全に同じオケを再現しながら、新しいひらがなで歌わせます。

### 🎲 モード3: 再生成モード (アレンジランダム)
歌詞は修正したものを使いつつ、メロディーやアレンジは**全く新しい別のパターン**を聴いてみたい場合はこのモードを選んでください。
プロンプトは維持したまま、シード値だけがランダムに変更されて新しい曲が生まれます。

## 🌟 高品質な曲を生み出す「黄金法則」

Ollama（作詞・作曲AI）に最高の仕事をさせるためには、`song_theme.txt` に書くテーマ文にコツがあります。

### ❌ 避けるべき書き方（ダメな例）
* 「アップテンポな曲」
* 「楽しいポップス」
*(※短すぎると、AIが月並みで退屈なプロンプトしか作れません)*

### ⭕ 理想的な書き方（良い例）
* 「80年代の日本のシティ・ポップ風。夏の夜の高速道路を走るドライブ。都会のネオンが流れていく。ベースがスラップでグルーヴ感を出し、ボーカルは少し大人っぽく透き通るような女性の声。」
*(※情景、時代感、使ってほしい楽器、ボーカルの性別や声質を具体的に書きます)*

**💡 コツ:** あなたは「映画監督」です。AIにどんなシーンのどんなBGMを作ってほしいのか、映像が目に浮かぶように情景を伝えてください。

## 🛠️ トラブルシューティング

**Q. ターミナルがすぐに閉じてしまって生成されない**
A. `song_theme.txt` の中身が空になっているか、保存されていない可能性があります。

**Q. 再生成モードを選んだのに「エラー：歌詞ファイルが指定されていません」と出る**
A. `song_theme.txt` にファイル名ではなく通常のテーマ文（「夜のドライブ…」など）が書かれています。再生成モードを使う時は、必ず `Lyrics_〇〇.txt` のファイル名を `song_theme.txt` にコピペして保存してください。

**Q. ボーカルの声が途中で小さくなったり、不鮮明になる**
A. SFTモデル（高音質）を選択して生成し直してみてください。Turboモデルは高速ですが、複雑なプロンプトではボーカルが不安定になることがあります。
